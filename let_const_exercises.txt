1. While you can redeclare a variable with var, you cannot with let. Var has a function scope while let has a block scope2. You cannot redeclare or reassign with const and const also has a block scope3. Let allows reassignment while const does not4. When your code is compiled, variable names declared with var are read first as if they were at the top of the page before they are defined. This is not the case with let or const.
